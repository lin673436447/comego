# TabBar 图标说明

请在此目录下放置以下图标文件（建议尺寸：81px * 81px）：

## 需要的图标文件：

1. **home.png** - 首页图标（未选中）
2. **home-active.png** - 首页图标（选中）
3. **record.png** - 记账图标（未选中）
4. **record-active.png** - 记账图标（选中）
5. **bill.png** - 账单图标（未选中）
6. **bill-active.png** - 账单图标（选中）
7. **stats.png** - 统计图标（未选中）
8. **stats-active.png** - 统计图标（选中）
9. **profile.png** - 我的图标（未选中）
10. **profile-active.png** - 我的图标（选中）

## 图标要求：

- 格式：PNG
- 尺寸：81px * 81px（推荐）
- 颜色：未选中状态使用灰色（#6B7280），选中状态使用蓝色（#1E40AF）
- 背景：透明

## 临时解决方案：

如果暂时没有图标，可以使用以下方式：

1. 使用在线图标生成工具（如：iconfont.cn）
2. 使用设计工具（如Figma、Sketch）创建简单图标
3. 下载现成的图标库

## 注意：

微信小程序对tabbar图标有严格要求，必须提供图标文件才能正常运行。
